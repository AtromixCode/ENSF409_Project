Logistics & Information General Management Application
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
This is a store manager application that allows clients to add, search for,
sell, remove, and order items in a store.
The application also allows items to be loaded or added to the store.
The application also allows suppliers to be loaded or added to the store.

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
The server must be running before running the client application is running.

To run the customer application, run the main method in the 
class "GUIController.java"
