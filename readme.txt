"Specialized for Users - General Management Application"
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
This is a customer application that allows customers to select items, add them
to their cart, and checkout.

Doing so will appropriately buy up items from the main store.

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
The server must be running before running the customer application is running.

To run the customer application, run the main method in the 
class "GUIController.java"
